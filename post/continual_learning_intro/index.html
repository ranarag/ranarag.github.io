Definition

A continual learning system can be defined as an adaptive algorithm capable of learning from a continuous stream of information, with such information becoming progressively available over time and where the number of tasks learned are not pre-defined. Critically, the accomodation of new information should occur without catastrophic forgetting or interference

Overwiew

Basically Continual Learning Involves:

  1. Ability to continually acquire, fine-tune, and transfer new knowledge and skills
  2. Higher and realistic time scale where data(and tasks) become available only with time.
  3. Minimal or no access to previously encountered data.
  4. Constrained computation and Memory resources.

Features

In continual learning the learning envrionment is non-stationary, divided into a sequence of learnable tasks available over time.

There are many variations:

  1. Tasks transitions can be smooth or discrete
  2. Length of tasks can vary and tasks might repeat as well.
  3. Type of incoming tasks can vary.
  4. There can be scenarios where tasks may not be well-defined.

Catastrophic Forgetting

Artifical Neural Networks(ANN) tend to forget the previously learned information upon learning new information. This tendency of ANNs is called catastrophic forgetting, catastrophic since this hampers the model’s performance to a great extent. A naive way to prevent this from happening is by clubbing the previous training with the new training data every time the model is to be trained for a new task. This clubbing of new training data with the old one renders the old knowledge gathered by the model useless simultaneously increasing the training time of the model.

For overcoming catastrophic forgetting, learning systems, must on one hand, show the ability to acquire new knowledge and refine existing knowledge on the basis of continuous input and, on the other hand, prevent novel input from significantly interfereing with the existing knowledge. This results in what is known as the stability Plasticity Dilemma.

Stability-Plasticity Dilemma

The extent to which a system must be plastic in order to integrate novel information and stable in order not to catastrophically interefere with consolidate knowledge is known as the stability-plasticity dilemma.

Approaches

Existing Continual Learning Approaches can be broadly classified into three categories:

  • Regularization Based
  • Dynamic Architectures
  • Complementary Learning System and Memory Replay

Regularization Based

Regularization based approaches alleviate catastrophic forgetting by imposing constraints on the update of the parameters of the neural networks. In case of over-parameterized neural networks, such approaches usually exploit the previously unused parameters to learn a new task.

Dynamic Architectures

These approaches change the architectural property of the network to accomodate new information.

Complementary Learning System and Memory Replay

These approaches make use of a small fixed size memory of instances of previous tasks which it replays along with the new instances during training a new task. This helps in interplay of an epoisodic memory(specific experience) of the current task and a semantic memory (general structure knowledge) from the instances of previous tasks stored in the memory to consolidate knowledge.
